🔐 Admin PIN Setup

How to set your action PINs:

1. Go to Render dashboard
2. Select your Sabicars-backend service
3. Click Environment
4. Add these variables:
  • INVENTORY_PIN = your chosen PIN
  • BLOG_PIN = a different PIN
5. Click Save Changes

After Render redeploys, anyone clicking Edit or Delete will need to enter the correct PIN first. PINs are remembered for the current browser session.
